# Kaxt Meyaj 📱

Kaxt Meyaj es una aplicación móvil enfocada en conectar personas con oportunidades laborales locales en Quintana Roo mediante una plataforma moderna, accesible y fácil de usar.

Este proyecto está siendo desarrollado como parte de Innovatec y tiene como objetivo apoyar a las comunidades locales mejorando el acceso a oportunidades de empleo.

---

## 🚀 Tecnologías utilizadas

- React Native
- Expo
- TypeScript
- Expo Router
- Firebase *(próximamente)*

---

## 📱 Funcionalidades actuales

- Pantalla principal
- Navegación entre pantallas
- Interfaz de inicio de sesión
- Interfaz de registro
- Listado de vacantes
- Componentes reutilizables
- Diseño responsive

## 🔥 Funcionalidades futuras

- Autenticación con Firebase
- Base de datos en tiempo real
- Postulación a vacantes
- Sistema de favoritos
- Perfil de usuario
- Publicación de empleos
- Filtros de búsqueda
- Soporte para idioma maya
